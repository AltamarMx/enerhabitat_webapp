# EnerHabitat Python
Herramienta para evaluar el desempeño térmico de un muro o techo en cualquier parte de la república con o sin uso de A/C

Creación de la interfaz Ener-Habitat, desarrollado con Shiny.

#Comandos para ejecucion Ingrese a la terminal de la carpeta clonada e ingrese el siguiente comando para correr la app.py: python -m shiny run --port 34799 app.py
